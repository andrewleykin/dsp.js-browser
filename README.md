# dsp.js-browser

A browser version of dsp.js 

See: https://github.com/corbanbrook/dsp.js

Exposes DSP as a global variable. 

Usage: 

Include the script: 

    <script src="/dist/DSP.js"></script>

Use the methods, e.g. DFT: 

var ft = new DSP.DFT(waveValues.length);
